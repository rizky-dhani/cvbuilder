# Technology Stack - CV Builder

## Core Backend
- **Language:** PHP 8.4
- **Framework:** Laravel 12.x
- **Database:** MariaDB
- **Tooling:** Laravel Sail (Docker-based development environment)

## Frontend & UI
- **Framework:** Livewire (Server-driven UI)
- **Styling:** Tailwind CSS 4.x
- **Bundler:** Vite
- **Admin/Dashboard:** Filament (For rapid development of management interfaces)

## Quality & Development
- **Testing:** Pest (Unit and Feature testing)
- **Formatting:** Laravel Pint
- **Debugging:** Laravel Pail, Laravel Tinker, Mailpit (Local email testing)
- **Assistant:** Laravel Boost MCP (For specialized Laravel & Database operations)
